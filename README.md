# XamDialogs

Custom dialogs for picking dates and from simple lists for Xamarin.iOS

 * Closely resembles the new style iOS 7+ AlertViews
 * Customisable
 * Date and Picker view in alert dialog
 * Extendable - create your own custom dialogs
 * Styleable
